
readme.txt

En este trabajo se calcula la energía potencial gravitacional
para el caso solar de acuerdo con el modelo de Christensen-
Dalsgaard.

	1. El archivo trabajo1.pdf contiene el planteamiento
	   del problema a resolver.
	2. El archivo cptrho.l5bi.d.15c contiene un conjunto
	   de datos limitado del modelo solar de Christensen-
	   Dalsgaard.
	3. El archivo Sun_GEP.ipynb es un cuaderno de Jupyter
	   en donde se resuelve el problema.
	   
	   Este cuaderno se ejecutó de forma local usando
	   Jupyter Notebook con Pyhton3.
	   
	   Además, este cuaderno puede visualizarse
	   directamente en Google Drive usando Google Colab.
	   Sin embargo, su ejecución en Google Colab no 
	   funcionará si el script no se modifica para que
	   lea el archivo cptrho.l5bi.d.15c desde Google
	   Drive.
